## How to Contribute

We welcome contributions from everyone! If you're interested in contributing to XJTU-ICS, please follow these steps üëÄ:

1. Fork the repository to your own GitHub account.
2. Clone the forked repository to your local machine.
3. Create a new branch for your changes.
4. Make your changes and commit them with clear and concise commit messages.
5. Push your changes to your forked repository.
6. Open a pull request to the main repository.
    - Since _Rule Set_ is in use, you must contribute with a PR!
7. Delete your branch after merging it. This keeps the repo clean and faster to sync.

It's always a good habit to check [CONTRIBUTING.md](https://github.com/xjtu-ics/textbook/blob/main/CONTRIBUTING.md) before contributing ‚ö†Ô∏è!
