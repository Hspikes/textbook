# Chapter 7 Linking

本章旨在深入剖析链接过程 —— 即编译器驱动如何调用链接器，将多个独立编译的代码和数据片段整合成一个单一的可执行文件。我们将研究静态链接和动态链接的机制，并理解链接器如何解析符号、重定位代码，以及它对现代软件开发的重要性。

    1. 静态链接
    2. 动态链接
    3. 库打桩机制
